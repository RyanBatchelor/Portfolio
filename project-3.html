Project Overview

Seek Geek is a work in progress. Seek Geek was originally made as a college project with HTML, CSS, Bootstrap, JavaScript, PHP and MYSQL. This recreation is trying to impove upon the old design that was made several years ago. Currently, users can create a profile, add friends and view their pages. Users can create posts with images. Users will only see posts from their friends. Users can like and comment on posts.
